## About

I operate at the intersection of software engineering, security automation, and infrastructure. My core stack is Python (FastAPI, Flask) for backend systems, complemented by React and Next.js when frontend work is needed. I design and build platforms that are secure by design, automated by default, and scalable from the start.

My work spans:

- **Backend engineering** — RESTful APIs, microservices, authentication/authorization, database design
- **Security automation** — vulnerability scanning pipelines, dependency analysis, threat detection
- **Offensive security** — reconnaissance tooling, penetration testing labs, attack simulation
- **Cloud & containerization** — Docker, CI/CD (GitHub Actions), Linux server management
- **Linux ecosystem** — Arch Linux, Bash scripting, system forensics
